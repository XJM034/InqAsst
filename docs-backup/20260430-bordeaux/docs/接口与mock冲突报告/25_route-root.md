# 接口与 Mock 冲突：`/`

## 1. 冲突等级

`低`

这不是正式业务页，冲突主要在默认入口逻辑。

## 2. mock 依赖的对象

mock 对根路由的期待很简单：

- 进入系统后能稳定去到正确入口

## 3. 当前接口实际使用的对象

当前根路由实际会受真实登录态、角色、校区等信息影响。

## 4. 具体冲突点

- 如果默认入口逻辑写死或依赖旧 mock 映射，会与真实登录态冲突。
- 如果根路由不消费真实登录结果，就可能把用户送到错误页面。

## 5. 处理建议

- 根路由只能基于真实登录态和真实角色结果做跳转。
- 不允许保留任何 mock 入口映射。
